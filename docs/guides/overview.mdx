## Hardware
### Storage (NAS/SAN)
For my NAS hardware im using [Dell PowerEdge R720xd](https://i.dell.com/sites/content/shared-content/data-sheets/en/Documents/Dell-PowerEdge-R720XD-Spec-Sheet.pdf) with following specifications :

|           |                                                                       |
|-----------|-----------------------------------------------------------------------|
| **CPU**   | 1x Intel Xeon E5-2660v2, 2.2GHz, 10C/20T                              |
| **RAM**   | 8x 32GB DDR3 ECC 1066 MHz 256GB                                       |
| **DISKS** | 4x WD DC HC520 12TB (mirror-pool0)                                    |
|           | 4x Toshiba N300 4TB (RAIDZ1-pool1)                                    |
|           | 4x Seagate EXOS 4TB (RAIDZ1-pool1)                                    |
|           | 2x Intel DCS3610 SSD 800GB (OS)                                       |
| **NIC**   | 2x 1GBase-T (I350)                                                    |
|           | 2x 10GBase-X (X520)                                                   |
| **FC HBA**| 1x Dual Port Emulex LPe12000 (Direct connec to my Tape Library )      |

It serves `block`, `file` and `object` storage services for my whole homelab environment

### Backup System
For my homelab backup system im using [IBM TS3200](https://www.redbooks.ibm.com/technotes/tips1342.pdf) tape library with following specifications:

|                     |                                                                       |
|---------------------|-----------------------------------------------------------------------|
| **DRIVE**           | 2x LTO-7 HH FC 8Gb Drive                                              |
| **TAPE**            | 24x LTO-6 Tapes (Partition 1)                                         |
|                     | 24x LTO-7 Tapes (Partition 2)                                         |
| **BACKUP SOFTWARE** | [Bacula](https://www.bacula.org/) Community Edition                   |

<Tip>
Offsites backup uses [Cloudflare R2](https://www.cloudflare.com/developer-platform/r2/)
</Tip>
